# 📊 Real-Time Stock Market Dashboard

This is a Streamlit-based Python project that shows real-time stock market data and visualizes it using charts.

---

## 🚀 Features
- Live stock price data
- Line chart for price trends
- Candlestick chart for analysis
- Select time range (1mo, 3mo, 6mo, 1y)
- Simple web dashboard using Streamlit

---

## 🛠️ Installation & Setup

### 1. Clone the repository
git clone https://github.com/your-username/stock-dashboard.git

### 2. Go to project folder
cd stock-dashboard

### 3. Create virtual environment
python -m venv venv

### 4. Activate virtual environment
venv\Scripts\activate

### 5. Install requirements
pip install -r requirements.txt

### 6. Run the app
streamlit run app.py

---

## 🌐 Open in browser
http://localhost:8501

## 🧠 Tech Stack
Python, Streamlit, yfinance, Pandas, Plotly
